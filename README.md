# JS-Counter
Pressing the [-1] button reduces the number by -1, and pressing the [+1] button increases the number by +1.

------------

+1 버튼을 클릭 시, 위에 있던 0이 1씩 커지고, -1 버튼을 클릭 시, 위에 있던 0이 -1씩 작아진다.
